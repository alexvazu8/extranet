Operador Descripción
+ Operador de suma. Se utiliza para sumar dos valores o concatenar cadenas.
- Operador de resta. Se utiliza para restar dos valores.
* Operador de multiplicación. Se utiliza para multiplicar dos valores.
/ Operador de división. Se utiliza para dividir un valor por otro.
% Operador de módulo. Se utiliza para obtener el resto de la división.
++ Operador de incremento. Se utiliza para incrementar en 1 el valor de una variable.
-- Operador de decremento. Se utiliza para decrementar en 1 el valor de una variable.
== Operador de igualdad. Se utiliza para comparar si dos valores son iguales.
!= Operador de desigualdad. Se utiliza para comparar si dos valores son diferentes.
< Operador de menor que. Se utiliza para comparar si un valor es menor que otro.
> Operador de mayor que. Se utiliza para comparar si un valor es mayor que otro.
<= Operador de menor o igual que. Se utiliza para comparar si un valor es menor o igual que otro.
>= Operador de mayor o igual que. Se utiliza para comparar si un valor es mayor o igual que otro.
&& Operador lógico AND. Se utiliza para evaluar si dos condiciones son verdaderas.
|| Operador lógico OR. Se utiliza para evaluar si al menos una de dos condiciones es verdadera.
! Operador lógico NOT. Se utiliza para negar una condición.
?: Operador Elvis. Se utiliza para asignar un valor por defecto si una variable es nula.
?. Operador de navegación segura. Se utiliza para acceder a una propiedad de un objeto de forma segura si el objeto no es nulo.
